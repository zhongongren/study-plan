# 初中生一个月学习计划表 — 内容框架

## 目标学生画像
- 年级：初中（初一/初二通用）
- 特点：基础较差，缺乏复习习惯，注意力难以集中
- 已报培训班，有配套 App（含语数英等课程）
- 需要吸引力强、可执行的计划

---

## 时间安排原则

### 周一至周五（工作日）
- 19:00 到家
- 19:00 - 21:30 学习时间窗口
- 21:30 之后准备上床休息

| 时间段 | 内容 | 说明 |
|--------|------|------|
| 19:00 - 19:10 | 到家放松 + 整理书包 | 缓冲过渡 |
| 19:10 - 20:10 | 完成当天学校作业 | 先把作业写完（约60分钟）|
| 20:10 - 20:20 | 休息 / 喝水 | 短暂放松 |
| 20:20 - 20:50 | 培训班 App 学习（30分钟）| 看当天指定课程视频 |
| 20:50 - 21:20 | 当天知识点复习（30分钟）| 回顾当天学校所学内容 |
| 21:20 - 21:30 | 整理错题本 / 明日计划 | 5分钟记录 + 5分钟预习 |
| 21:30 | 上床休息 | 保证充足睡眠 |

### 周六至周日（全天）
- 全天可用于学习
- 合理安排休息，避免疲劳

| 时间段 | 内容 | 说明 |
|--------|------|------|
| 08:00 - 08:30 | 起床 + 早餐 | |
| 08:30 - 09:30 | 上周知识点总复习（60分钟）| 重点薄弱科目 |
| 09:30 - 10:30 | 培训班 App 课程（60分钟）| 数学 or 语文 |
| 10:30 - 10:45 | 休息 | |
| 10:45 - 11:45 | 培训班 App 课程（60分钟）| 英语 or 其他科目 |
| 11:45 - 13:30 | 午饭 + 午休 | |
| 13:30 - 14:30 | 专项练习（60分钟）| 做题巩固 |
| 14:30 - 15:30 | 错题整理 + 笔记整理（60分钟）| 建立错题本 |
| 15:30 - 15:45 | 休息 | |
| 15:45 - 16:45 | 自由学习时间（60分钟）| 预习下周内容 or 感兴趣的内容 |

| 17:00 之后 | 自由活动 + 晚饭 | |
| 晚上（可选）| 轻松复习（30分钟）| 看看笔记，不做题 |

## 一个月科目轮换计划（4周）

### 第1周：打基础
- 重点科目：数学（基础运算）、语文（字词积累）
- App 课程：每天1节，以基础课为主
- 复习重点：上学期遗留的基础知识

### 第2周：攻英语
- 重点科目：英语（单词 + 语法基础）
- App 课程：英语专项，每天背10个单词
- 复习重点：第1周内容 + 英语基础

### 第3周：综合提升
- 重点科目：数学（解题方法）、语文（阅读理解）
- App 课程：综合课程，查漏补缺
- 复习重点：第1-2周内容

### 第4周：总结冲刺
- 重点科目：全科复习
- App 课程：错题讲解课
- 复习重点：整理一个月错题本，做一套综合练习题

---

## 激励机制（让计划表更有吸引力）

1. **打卡积分制**：每完成一个时间段打一个✓，一周集满25个✓可以换取奖励
2. **周目标小旗帜**：每周设定1个小目标，完成后贴上星星贴纸
3. **进度条可视化**：在计划表上设计进度条，看到自己的进步
4. **颜色区分科目**：不同科目用不同颜色标注，视觉清晰
5. **正能量口号**：每周一句激励语

## 设计风格
- 色彩：活泼但不杂乱，以蓝色/绿色/橙色为主
- 字体：清晰易读，标题醒目
- 版式：月历 + 每日时间表双版面
- 可打印 A4 尺寸
